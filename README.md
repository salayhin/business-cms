Business CMS
============

Complete CMS for small business. Business CMS based on ruby on rails 3.2.13 and postgresql.

Gem Used:
--------
* gem 'pg'
* gem 'devise', '3.2.2'
* gem 'carrierwave', ' 0.9.0'
* gem 'rmagick', '~> 2.13.1'
* gem 'activeadmin', '~> 0.6.3'
* gem 'ckeditor', '~> 4.0.10'
